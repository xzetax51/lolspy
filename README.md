# 🕵️ Шпион: LoL Edition

**Telegram Mini App** — классическая игра «Шпион» (Spyfall) в стилистике **League of Legends**.

Игроки получают карточки с чемпионами LoL. Все видят одного и того же чемпиона — кроме **Шпиона**, который получает карточку «Шпион». Задача мирных — вычислить шпиона через обсуждение и голосование. Задача шпиона — выжить и угадать общий чемпион по ответам других.

---

## ✨ Возможности

- 🎮 **Лобби по коду** — создай комнату, поделись 5-символьным кодом с друзьями
- 👥 **От 3 игроков** — минимум для старта, максимум не ограничен
- 🗡️ **167 чемпионов LoL** — полный актуальный ростер с официальной русской локализацией
- 🎲 **Случайное распределение** — шпион и чемпион выбираются случайно
- 📋 **Порядок ходов** — шпион никогда не говорит первым (честная игра!)
- 🗳️ **Голосование** — выбирай подозреваемого или пропусти
- 📊 **Экран результатов** — кто шпион, какой был чемпион, таблица голосов
- 🔄 **Новый раунд** — играй с тем же составом без пересоздания лобби
- 📱 **Telegram Mini App** — работает прямо в Telegram, не нужно ничего устанавливать

## 🚀 Быстрый старт (локально)

### 1. Клонируйте репозиторий
```bash
git clone https://github.com/ваш-username/lolspy.git
cd lolspy
```

### 2. Установите зависимости
```bash
pip install -r requirements.txt
```

### 3. Запустите сервер
```bash
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

### 4. Откройте в браузере
```
http://localhost:8000
```

> ⚠️ **Важно:** Локально приложение работает с in-memory хранилищем. При перезапуске сервера все активные лобби будут потеряны. Для продакшена используйте Redis (см. раздел «Деплой»).

## ☁️ Деплой на Vercel + Upstash (бесплатно, без карты)

Это рекомендуемый способ запуска. Vercel — serverless-платформа, Upstash — бесплатный Redis.

### Шаг 1. Создайте базу данных Upstash
1. Зарегистрируйтесь на [upstash.com](https://upstash.com/) через GitHub (карта не нужна)
2. Нажмите **Create Database** → выберите регион (например, Frankfurt)
3. Откройте базу → **Connect** → скопируйте строку `rediss://default:...` (именно с двумя `s`!)

### Шаг 2. Задеплойте на Vercel
1. Зарегистрируйтесь на [vercel.com](https://vercel.com/) через GitHub
2. Нажмите **Add New → Project** → выберите репозиторий
3. Vercel автоматически определит Python-проект. Нажмите **Deploy**

### Шаг 3. Настройте переменные окружения
В **Settings → Environment Variables** добавьте:

| Переменная | Значение |
|------------|----------|
| `BOT_TOKEN` | Токен от [@BotFather](https://t.me/BotFather) |
| `WEBAPP_URL` | URL вашего Vercel-приложения (например, `https://lolspy.vercel.app`) |
| `UPSTASH_REDIS_URL` | Строка `rediss://...` из Upstash |

### Шаг 4. Перезапустите деплой
1. Во вкладке **Deployments** нажмите `...` → **Redeploy**
2. Это нужно, чтобы бот отправил в Telegram правильный webhook URL

### Шаг 5. Настройте бота
1. Откройте [@BotFather](https://t.me/BotFather)
2. `/mybots` → выберите бота → **Bot Settings** → **Menu Button** → вставьте URL вашего Mini App
3. Готово! Теперь бот будет выдавать кнопку «🎮 Открыть игру»

## 🎮 Как играть

1. **Хост** открывает Mini App через бота и нажимает «Создать лобби»
2. Получает 5-символьный код (например, `X7K2P`)
3. Делится кодом с друзьями — они вводят его в приложении
4. Когда собрались 3+ игрока, хост нажимает **«Начать игру»**
5. Каждый видит свою карточку:
   - Мирные жители видят чемпиона (например, **«Ясуо»**)
   - Шпион видит карточку **«Шпион»**
6. Игроки по очереди задают вопросы и отвечают (шпион не первый!)
7. Хост запускает голосование — каждый выбирает подозреваемого
8. После голосования показываются результаты: кто шпион, какой был чемпион, кто пойман
